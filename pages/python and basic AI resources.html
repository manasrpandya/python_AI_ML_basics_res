Additional Resources - ChatGPT/Perplexity/Google

Apart from learning through sites and books and videos, there is equal (if not more) scope for learning through quick searches on LLM's like ChatGPT, or simply Googling (a smarter version is Perplexity, which I find myself using more often). Although, this has certain good ways to learn and certain bad ways to learn from such online automated resources.

How and How NOT to use ChatGPT (and Google, Perplexity, etc.):

ChatGPT (an online free-to-use LLM) is a great tool to learn many different things, and it should be; considering it was trained on, essentially, the entire Internet. For effective learning, however (especially to learn Python, AI), Users shouldn't simply search for the answer to a specific question and copy paste it to their assignments/code environments. There must be active learning involved. A good learner should always first read and try to understand the answer, replicate it him/herself, and ask ChatGPT to judge their replication iteratively.

As far as coding is concerned, always ask the LLM to use lots of comments/explanations to enhance the learning of the user. Sure, when you're working on a deadline, you might want to skip learning. But whilst learning, always focus on YOUR learning, instead of providing answers to a question mindlessly.

When to use LLM's?

Use LLM's only after first searching on existing work. You'll always find help online through sites like reddit, quora, stack exchange and github. This isn't to say that you should use LLM's as a last resort. You should side-by-side try to extrapolate ideas from the afore-mentioned sites using LLM's

Remember, ChatGPT isn't just available to you. It is open to everyone. Don't slow the process of your learning to just abide by the standards of learning set by the so called experts and professionals. If you can learn ideas, replicate work and recall concepts, you 'know' the subject. It doesn't matter which path you take - LLM's or Books/Videos.
